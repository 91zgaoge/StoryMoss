## 🎯 项目背景

### 项目概况
- **项目名**: StoryForge (草苔) v5.6.4 - AI 导演式小说创作系统
- **架构**: Tauri + Rust 后端 + React 前端
- **当前状态**: 功能层面100%，但存在18个关键实现问题
- **问题来源**: 代码库映射分析报告 (`.planning/codebase/`)

### 核心问题
通过全面代码库分析发现，项目在以下方面存在设计与实现差距：

1. **数据完整性风险** - SQLite外键约束未启用，级联删除不完整
2. **前端缓存同步失效** - 数据修改后前端不刷新
3. **自动化功能不闭环** - 工作流重启后不恢复，能力进化仅启动时触发
4. **代码质量问题** - 大量unwrap()调用，内存压力，维护性差
