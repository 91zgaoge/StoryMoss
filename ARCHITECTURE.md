> **v0.26.9 зЁіе®ҡжҖ§иЎҘдёҒ**пјҡеңЁ v0.26.8 еҹәзЎҖдёҠиҝӣдёҖжӯҘж №жІ» Genesis ж–°е°ҸиҜҙз¬¬дёҖз« еҶ…е®№йҮҚеӨҚй—®йўҳгҖӮйҮҚеӨҚжЈҖжөӢдёҺеүҚзјҖеҺ»йҮҚз»ҹдёҖж”№з”Ё `latestContentRef.current`пјҲReact state еҗҢжӯҘеҝ«з…§пјүпјҢйҒҝе…Қ TipTap DOM ж»һеҗҺеҜјиҮҙе·ІжңүжӯЈж–Үиў«еҶҚж¬ЎиҝҪеҠ жҲ–жҒўеӨҚдёәе№ҪзҒөж–Үжң¬пјӣ`RichTextEditor` е№ҪзҒөж–Үжң¬зӣҙжҺҘеҢ…еҗ«жЈҖжөӢеүҘзҰ» HTML ж ҮзӯҫпјҢиҰҶзӣ– ContentUpdate/AppendContent и·Ҝеҫ„гҖӮ

## жһ¶жһ„зҗҶеҝө

StoryForge йҮҮз”ЁеҲӣж–°зҡ„**еү§йҷўејҸеҸҢз•Ңйқўжһ¶жһ„ + еңәжҷҜеҢ–еҸҷдәӢ + еўһејәи®°еҝҶзі»з»ҹ**пјҡ

- **е№•еүҚ (Frontstage)**: жІүжөёејҸеҶҷдҪңз•ҢйқўпјҢеҰӮеҗҢзҷ»еҸ°жј”еҮә
- **е№•еҗҺ (Backstage)**: дё“дёҡе·ҘдҪңе®ӨпјҢеҰӮеҗҢеҗҺеҸ°еҮҶеӨҮ
- **еңәжҷҜ (Scene)**: жҲҸеү§еҶІзӘҒй©ұеҠЁзҡ„еҸҷдәӢеҚ•дҪҚпјҢеҸ–д»Јдј з»ҹз« иҠӮ
- **и®°еҝҶ (Memory)**: еҹәдәҺ llm_wiki зҡ„зҹҘиҜҶеӣҫи°ұпјҢзңҹжӯЈзҡ„"и¶ҠеҶҷи¶ҠжҮӮ"

v0.25.0 еңЁеҺҹжңүжҷәиғҪеҲӣдҪңй“ҫи·Ҝд№ӢдёҠеўһеҠ дәҶдёӨйҒ“йҳІеҫЎжҖ§еҹәзЎҖи®ҫж–Ҫпјҡ

#### 1. Context Rot жҳҫејҸйҳІеҫЎ

й•ҝзі»з»ҹжҸҗзӨәиҜҚе®№жҳ“еҮәзҺ° "Lost in the Middle"вҖ”вҖ”жЁЎеһӢеҜ№дёӯй—ҙж®өиҗҪзҡ„зәҰжқҹи®°еҝҶиЎ°еҮҸгҖӮ`ContextPrioritizer` жҠҠзі»з»ҹжҸҗзӨәиҜҚжӢҶеҲҶдёәеёҰдјҳе…Ҳзә§зҡ„ `ContextChunk`пјҡ

- **Critical**пјҡеҗҲеҗҢзәўзәҝгҖҒеңЁдё–дҪңиҖ…дҝқжҠӨгҖҒеҸҚ AI йҷҲиҜҚж»Ҙи°ғзӯүз»қеҜ№дёҚиғҪиҝқеҸҚзҡ„зәҰжқҹ
- **High**пјҡи§’иүІж ёеҝғгҖҒдјҸз¬”гҖҒеӣӣе…ғз»„зӯүејәдҝЎеҸ·дёҠдёӢж–Ү
- **Normal**пјҡеҶҷдҪңйЈҺж јгҖҒж–№жі•и®әгҖҒдё–з•Ңи®ҫе®ҡзӯүеёёи§„дёҠдёӢж–Ү
- **Background**пјҡеҺҶеҸІеҸӮиҖғгҖҒејұе…іиҒ”и®°еҝҶзӯүиҫ…еҠ©дҝЎжҒҜ

жҺ’еәҸеҗҺ Critical дҝЎжҒҜеҗҢж—¶еҮәзҺ°еңЁжҸҗзӨәиҜҚејҖеӨҙе’Ңз»“е°ҫзҡ„иҪ»йҮҸж‘ҳиҰҒдёӯпјҢе®һзҺ°еҸҢйҮҚй”ҡе®ҡгҖӮ`ContextHealthMetrics` и®°еҪ•еҗ„зұ»еҲ«зҡ„ token ж•°дёҺйў„з®—дҪҝз”ЁзҺҮпјҢдҫӣиҜҠж–ӯеҚЎзүҮе®һж—¶жҹҘзңӢгҖӮ

#### 2. еӣӣзә§й”ҷиҜҜеҲҶзұ»дёҺжҒўеӨҚ

еҗҺз«Ҝй”ҷиҜҜз»ҹдёҖжіЁе…Ҙ `ErrorSeverity`пјҡ

| Severity       | еҗ«д№ү         | е…ёеһӢеңәжҷҜ              | жҒўеӨҚзӯ–з•Ҙ                                    |
| -------------- | ------------ | --------------------- | ------------------------------------------- |
| **Fatal**      | ж— жі•иҮӘеҠЁжҒўеӨҚ | жңӘзҹҘеҶ…йғЁ panic        | и®°еҪ•е№¶з»ҲжӯўпјҢеүҚз«Ҝеј№ `AgentInterruptionModal` |
| **Retry**      | зһ¬жҖҒй”ҷиҜҜ     | жЁЎеһӢиҝһжҺҘи¶…ж—¶гҖҒDB й”Ғе®ҡ | жҢҮж•°йҖҖйҒҝйҮҚиҜ•пјҢжҲ–еҲҮжҚўеҖҷйҖүжЁЎеһӢ                |
| **Degraded**   | еҸҜйҷҚзә§з»§з»ӯ   | дёҠдёӢж–ҮзјәеӨұ            | жү§иЎҢз®ҖеҢ–и·Ҝеҫ„е№¶жҸҗзӨәз”ЁжҲ·                      |

    pub async fn ingest(&self, content: &IngestContent) -> Result<(), Error> {
        // Step 1: еҲҶжһҗйҳ¶ж®ө
        let analysis = self.analyze_content(content).await?;
        // жҸҗеҸ–пјҡе®һдҪ“гҖҒе…ізі»гҖҒдәӢд»¶гҖҒжғ…ж„ҹгҖҒдјҸз¬”

        self.save_to_vector_store(&knowledge).await?;
